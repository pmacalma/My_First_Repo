Role: UX/UI Designer

Team: Phillip Macalma & Mike Gomez

Software: Figma, Illustrator, Miro, Photoshop

Problem: Website had no real navigation and it was difficult to figure out where to find certain things.

City of Stockton Parks and Recreations

We created a modern and streamlined design that's easy to navigate and visually appealing. Some of the features we included were:

  • Simplified menus and navigation that make it easy to find the information you need quickly.
  • A responsive design that works seamlessly on desktops, tablets, and mobile devices, so youcan access our site from anywhere.
  • Improved search functionality that helps you find what you're looking for faster and more accurately.

User Testing

We came up with 3 tasks for the users to complete. We tested 5 users and all of them found it hard to navigate through the site and correctly get the tasks completed.

Annotations

We took annotations to see where we could improve the website. Here were some of the ideas we came up with:

  • Hero section that has upcoming events below it so that residents can stay informed and up to date
  • Add more photos to break up all the text
  • Create cards for parks, programs and activities so the feel of the website is more cohesive
  • Clean up navigation and organize information so that it is easier to find

Prototyping

We came up with a few pages that we thought we make the website easier to navigate and make the site more visually pleasing and better aligned.

Hi-Fidelity Prototype

We did some user testing on our prototypes and we made slight changes and came up with the our final hi-fidelity prototype.
